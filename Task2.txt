Задача А
Необходимо получить полный список файлов с переводом и выстроить их в алфавитном порядке по языку, на который переведено содержание файла, для последюущего удобства работы с создающимся Excel файлом.
Для каждого языка создаются отдельные листы в Excel файле и в них создавать таблицу с текстовыми строками
1 строка будет храниться в 1 ячейке, размер которой будет соответствовать содержимому. Строка будет иметь высоту самого большоего элемента содержащегося в ней.
Строки с переводом будут находится напротив источника в виде: Источник - Перевод.
Таблицы в каждом листе стоит делить по принадлежности к пространствам имен таким как: "HideAndSeek", "PartyFood". И дополнительно по принадлежности к говорищему, к примеру "Children".
Для удобства работы с файлом стоит создать класс, включающий поля: Source, Translation, Key

Задача Б
Решение данной задачи будет включать в себя все шаги из задачи А
Для того чтобы не проверять весь файл на наличие изменений стоит сразу запоминать ключ измененного элемента
Для изменения текста в loc файле необходимо будет находить элемент по ключу и записывать изменения
Сохранение изменений будет происходить при закрытии файла Excel или нажатии кнопки/выполнения комманды save
